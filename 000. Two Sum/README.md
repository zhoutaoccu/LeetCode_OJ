#0 ：暴力循环匹配
for循环两次，然后遍历所有的两两组合，然后判断是否满足 `a + b = target`，满足返回其下标。

#1：kv结构
这道题有点像找配对。找到 `a`，找到 `b`，让 `a + b = target`。那么肯定得遍历，遍历的过程中，记录什么，成了思考的关键。

既然是配对，那么 kv 结构是非常合适的，于是上了 Hash 表。让 key 就是要找的 `b`，让 value 记录 `a` 的 index，也就是结果需要的索引。
